# Services

The four AI-LinkMO access patterns:

* [AI-LinkMO CLI](cli.md) - command-line access for automation and CI/CD (fastCLI/slowCLI).
* [FastAPI Backend](fastapi-backend.md) - REST API generated dynamically from the OpenAPI spec.
* [Svelte Web UI](web-ui.md) - Vite + Svelte 5 SPA for non-technical exploration.
* [Neo4j Graph Database](graph-db.md) - relationship analysis over the exported knowledge graph.
