# Overview

`byo/data/owasp_llm_top_10.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus Container YAML (RiskTaxonomy + Risk rows); built from docs/_data/owasp-llm.yml.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [OWASP Top 10 for LLM Applications reference](../references/owasp-llm-top-10.md). Validate against the ontology schema with `uv run linkml validate byo/data/owasp_llm_top_10.yaml -s <ai-risk-ontology.yaml>`.
