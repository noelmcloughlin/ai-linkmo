# Overview

`byo/data/finos-aigf.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus knowledge_graph YAML (importable upstream); built from the framework's docs/_risks/, docs/_mitigations/ and docs/_usecases/.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [FINOS AI Governance Framework reference](../references/finos-aigf.md). Validate against the ontology schema with `uv run linkml validate byo/data/finos-aigf.yaml -s <ai-risk-ontology.yaml>`.
