# Overview

Generated by the CLI: `./ai graph --export --byod` produces the merged knowledge graph as `graph/ai-risk-ontology.yaml` (~45k lines), and `./ai graph cypher --export --byod` produces `graph/cypher/ai-risk-ontology.cypher` via `graph/cypher/export.py` (cymple QueryBuilder over the linkml instance data). The Cypher distribution is what the [Neo4j graph database service](../services/graph-db.md) imports.

Both distributions are regenerable artifacts - the sources of truth are the upstream ai-atlas-nexus package data and the BYOD files under `byo/data/`.
