# Overview

`byo/data/nist_ai_600_1.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus Container YAML (RiskTaxonomy + Risk rows, generative-AI profile); built from docs/_data/nist-ai-600-1.yml.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [NIST AI 600-1 GenAI profile reference](../references/nist-ai-600-1.md). Validate against the ontology schema with `uv run linkml validate byo/data/nist_ai_600_1.yaml -s <ai-risk-ontology.yaml>`.
