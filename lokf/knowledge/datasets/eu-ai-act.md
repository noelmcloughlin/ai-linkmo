# Overview

`byo/data/eu_ai_act.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus Container YAML (Taxonomy + Requirement rows per article); built from the FINOS AIGF repo's docs/_data/eu-ai-act.yml.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [EU AI Act requirements reference](../references/eu-ai-act.md). Validate against the ontology schema with `uv run linkml validate byo/data/eu_ai_act.yaml -s <ai-risk-ontology.yaml>`.
