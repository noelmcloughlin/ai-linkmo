# Overview

`byo/data/iso_42001.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus Container YAML (RiskControlGroupTaxonomy + RiskControl rows, Annex A); built from docs/_data/iso-42001.yml.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [ISO/IEC 42001 controls reference](../references/iso-42001.md). Validate against the ontology schema with `uv run linkml validate byo/data/iso_42001.yaml -s <ai-risk-ontology.yaml>`.
