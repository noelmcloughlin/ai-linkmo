# Overview

`byo/data/ffiec_it_handbook.yaml` is a bring-your-own-data (BYOD) file consumed by the FastAPI backend, CLI and web UI alongside the upstream ai-atlas-nexus data.

**Format:** ai-atlas-nexus Container YAML (Taxonomy + Group rows per booklet + Documentation rows per section); built from docs/_data/ffiec-itbooklets.yml.

**Provenance:** AUTOGENERATED by the FINOS AIGF build script (`linkml/scripts/build_finos_data.py`) from content maintained in the [FINOS AI Governance Framework](https://air-governance-framework.finos.org/) repository; the encoded authority is the [FFIEC IT Handbook reference](../references/ffiec-it-handbook.md). Validate against the ontology schema with `uv run linkml validate byo/data/ffiec_it_handbook.yaml -s <ai-risk-ontology.yaml>`.
