# IBM

IBM publishes [ai-atlas-nexus](https://ibm.github.io/ai-atlas-nexus/), the Python package (a `pyproject.toml` dependency of this repository) that provides the [AI Risk Ontology](../references/ai-risk-ontology.md) LinkML schema and the upstream risk/taxonomy data served by every AI-LinkMO access pattern.
