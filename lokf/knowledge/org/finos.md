# FINOS

FINOS (Fintech Open Source Foundation) fosters open collaboration in financial services technology. It publishes the [AI Governance Framework](../references/finos-aigf.md), whose repository also curates the framework encodings (EU AI Act, ISO/IEC 42001, NIST profiles, OWASP top-10s, SR 11-7, FFIEC) that `linkml/scripts/build_finos_data.py` turns into this repository's `byo/data/*.yaml` files.
